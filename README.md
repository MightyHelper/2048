# 2048 Game

Simple [2048](https://en.wikipedia.org/wiki/2048_(video_game)) app using [Next.js](https://nextjs.org/).

## How to run locally

```bash
npm install
npm run dev
```

## How to run with docker compose

```bash
docker compose build
docker compose up
```

## Pull the image

```bash
docker pull ghcr.io/ailuros-code/2048:main
```
